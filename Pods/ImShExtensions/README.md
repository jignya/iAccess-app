# ImShExtensions

Useful extensions for UIViewController, UIView, UITableView and so on....

## Requirements

- iOS 8.0+ / macOS 10.10+ / tvOS 9.0+ / watchOS 2.0+
- Swift 4

### CocoaPods
Specify "ImShExtensions" in your ```Podfile```:
```ruby 
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'
use_frameworks!

pod 'ImShExtensions'
```
